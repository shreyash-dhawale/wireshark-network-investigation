## Analysis Report

This Wireshark lab captured live network traffic on Kali Linux and analyzed it using display filters, TCP stream following, and statistics views.

### Findings
- DNS traffic showed normal domain lookups.
- TCP handshakes followed standard SYN/SYN-ACK/ACK behavior.
- A few retransmissions were observed, likely due to minor network delay.
- TLS traffic showed normal encrypted session establishment.
- No clear malicious activity was identified in the capture.

### Conclusion
This project improved my ability to inspect packets, understand protocol behavior, and document network activity like a junior cybersecurity analyst.
